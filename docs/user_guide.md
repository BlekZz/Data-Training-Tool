# 📘 使用者操作指南 (User Guide)

歡迎使用 **Data Judgment Training Platform**！本平台旨在訓練你對資料的敏感度與清洗邏輯。

---

## 🚀 快速開始流程

### 0. 獲取個人專屬試算表
*   **步驟**: 點開管理員提供的「訓練平台範本 (Template)」。
*   **動作**: 點選左上角 `檔案 (File)` -> `建立副本 (Make a copy)`。
*   **注意**: 副本會連同後台的 Apps Script (User Side Logic) 一併複製，請勿刪除任何內建程式碼。

### 1. 初始化環境
*   點選上方選單：`🚀 數據判斷訓練` -> `1. 初始化 Home 頁面`。
*   若系統要求授權，請點選「進階」並允許存取。

### 2. 設定 API Key (重要！)
*   **獲取方式**: 至 [Google AI Studio](https://aistudio.google.com/app/apikey) 免費申請。
*   **設定**: 切換到 `Home` 分頁，在 **B4 儲存格** 貼上你的 API Key。
*   **警告**: API Key 僅儲存於你個人的這份 Sheet 中，不會被後端永久儲存，但請勿隨意分享你的 Sheet 給他人。

### 3. 生成新題目
*   點選上方選單：`🚀 數據判斷訓練` -> `2. 生成新題目`。
*   系統會跳出彈窗顯示「AI 題庫生成中...」，請耐心等待約 15-30 秒。
*   生成成功後，會自動建立一個名為 `Q_xxxx` 的新分頁。

---

## 🧠 核心任務說明

進入題目分頁後，你將看到三個主要區塊：

### 區塊 A：任務情境與原始資料
*   閱讀頂部的 **【任務情境】**，裡面隱藏了判斷資料正確性的邏輯線索。
*   檢查中間的 **Sample Data**，找出其中的格式錯誤或邏輯地雷。

### 區塊 B：步驟一 — 健檢診斷 SOP (淡黃色區塊)
*   將你發現的每個地雷填入表格。
*   **異常欄位**: 哪個欄位出錯？
*   **錯誤類型**: 選擇下拉選單 (如：基礎格式、商業邏輯)。
*   **處置策略**: 你打算如何處理 (如：刪除、修正、保留並註釋)。

### 區塊 C：步驟二 — 清洗後資料 (淡綠色區塊)
*   **輸入標題**: 在深綠色列輸入你清洗後想保留的欄位名稱。
*   **貼上資料**: 將你清洗過的數據貼入下方區塊。

---

## 📊 提交與評分

1.  確認 SOP 與清洗資料都填寫完畢。
2.  點選上方選單：`🚀 數據判斷訓練` -> `3. 提交本題回答`。
3.  確認提交後，AI 會開始進行深度評分。
4.  **查看結果**: 捲動到分頁下方，你會看到：
    *   **雷達診斷分數**: 包含格式、邏輯、策略、完整度。
    *   **AI 盲區評論**: 告訴你漏掉了哪些細節。
    *   **標準解答與範本**: 供你對照自己的不足。

---

## 🛠️ 疑難排解與 Debug 指南

| 錯誤現象 | 可能原因 | 解決方法 |
| :--- | :--- | :--- |
| **點選選單沒反應** | GAS 權限未授權 | 重新執行「初始化」並在彈窗點選允許。 |
| **生成題目顯示 API Error** | API Key 無效或額度耗盡 | 檢查 AI Studio 的 Key 是否正確，或檢查網路。 |
| **提交時顯示「找不到 SOP 區塊」** | 標題文字被修改 | 請確保「🧠 步驟一」的標題文字與位置未被更動。 |
| **評分後沒有顯示結果** | 網路中斷 | 檢查 `AuditLog` 或嘗試重新點選提交。 |
| **資料錯位** | 步驟二標題不完整 | 請確保所有清洗後的資料都有對應的標題，且標題列沒有斷層。 |

---

## ❓ 常見問題
*   **Q: 為什麼點擊生成沒反應？**
    *   A: 請檢查 `Home` 頁面的 API Key 是否填寫正確，且帳號是否有網路連線。
*   **Q: 提交後可以修改嗎？**
    *   A: 可以重新修改後再次提交，AI 會覆蓋舊的分數。
